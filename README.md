# CodeAlpha_BasicNetworkSniffer

## Internship Task 1
This project is part of the CodeAlpha Cyber Security Internship.

## Project Objective
The objective of this project is to build a basic network sniffer using Python and Scapy to capture and analyze network traffic.

##  Technologies Used
- Python 3
- Scapy Library
- Kali Linux

## Features

- Captures live network packets
- Displays Source and Destination IP
- Identifies Protocol (TCP, UDP, ICMP)
- Displays Port numbers
- Real-time traffic monitoring

##  Concepts Covered
- Network Packets
- IP Addressing
- TCP/UDP Protocols
- ICMP (Ping)
- Packet Inspection
